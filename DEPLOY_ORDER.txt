NAAM JAAP COUNTER v2.6 — OFFLINE SAFE DEPLOYMENT ORDER

IMPORTANT: DEPLOY THE APPS SCRIPT BACKEND FIRST.
The new frontend sends stable operation IDs. The backend patch is required to safely de-duplicate completed offline batches.

1. GOOGLE APPS SCRIPT
   - Replace existing 01_Server.gs contents with:
     apps-script/01_Server_SECURE_REPLACEMENT.gs
   - Confirm these existing secure files are present/current:
     apps-script/06_Auth.gs
     apps-script/07_UserActivity.gs
   - Add the new file:
     apps-script/08_OfflineSync.gs
   - Replace/update:
     apps-script/99_Auth_Setup.gs
   - Save all files.

2. RUN SETUP
   - Select setupSecureAuthentication() and Run it once.
   - Approve permissions if requested.
   - Confirm these sheets exist:
     Users
     Devices
     User_Activity
     Sync_Operations

3. DEPLOY A NEW WEB APP VERSION
   Deploy > Manage deployments > Edit > New version > Deploy

   Keep:
   Execute as: Me
   Who has access: Anyone

   The /exec URL normally stays the same when an existing deployment is edited.

4. TEST BACKEND
   Open the /exec URL. It should return JSON containing:
   version: 2.6.0-offline-safe
   authRequired: true

5. GITHUB PAGES FRONTEND
   - Upload the frontend files from the package root to the repository root.
   - Do not upload the apps-script folder publicly unless intended.
   - Commit changes and wait for Pages deployment.

6. CLEAR OLD PWA CACHE
   Desktop: Ctrl + Shift + R
   Mobile: clear site data or uninstall the old PWA, open the website once online, then install it again.

7. OFFLINE TEST
   - Sign in online once.
   - Tap several counts and wait for Synced.
   - Turn on Airplane Mode.
   - Reopen the installed app.
   - Tap at least 5 times; status should say saved locally.
   - Fully close and reopen; the count should remain.
   - Restore internet.
   - Tap Sign in to Sync and authenticate.
   - Confirm Sync_Operations and User_Activity entries and the Dashboard total.

SECURITY NOTES
- Never put a Google OAuth Client Secret in GitHub or Apps Script frontend files.
- Google ID tokens remain in sessionStorage only.
- Offline access expires after 7 days without a fresh verified login.
- Explicit Sign Out removes offline access from that device.
